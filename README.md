# UTS Trip Tracking Service
This is a simple web application for tracking trips and bookings. It consists of a MySQL database, a Node.js server, and a frontend interface.

Trip Tracking Service: An application that allows users to track their upcoming trips from one place. 
User can view all upcoming trips shown on the page.
User can choose a domestic origin + destination and add them as part of a future trip.
User can select trips from the Upcoming Trips table and edit the details or remove them from the list.
